# SalesTaxes

Simple application to print out the receipt for shopping baskets.

```
Usage: java -jar SalesTaxes-1.0-SNAPSHOT.jar [options] Input files
  Options:
    --help

    --importTax, -i
      The % of the import tax
      Default: 5
    --salesTax, -s
      The % of the sales tax
      Default: 10
```
